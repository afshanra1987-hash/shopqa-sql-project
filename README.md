# ShopQA SQL Project

## Overview

This project demonstrates SQL skills used in Manual QA testing for an e-commerce application.

The project includes:

- Database design
- Table creation
- Primary & Foreign Keys
- Data insertion
- SQL queries
- Filtering
- Sorting
- Aggregate functions
- INNER JOIN
- API Testing
- Bug Reporting
