# ShopQA SQL Project

## Overview

This project demonstrates SQL skills by designing and querying a simple e-commerce database using MySQL.

The project includes:

- Database design
- Table relationships
- Primary Keys
- Foreign Keys
- Sample data
- SQL queries
- INNER JOIN
- Aggregate functions

---

## Database Structure

The database contains four tables:

- Users
- Products
- Orders
- Order Items

---

- CREATE TABLE

- COUNT()

- LIKE
